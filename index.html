Sobre o starterkit

AdlayerStarter kit é um template/tutorial para ajudar você a entender como a integração com Adlayer funciona.

Sua simples missão:

Mostrar uma peça publicitária no bloco abaixo:

Siga os seguintes passos:

  1. Adicionando o site
    • Faça login em sua conta Adlayer
    • Adicione um site com o nome "starterkit"
  2. Integrando a página
    • Dentro do site "starterkit", adicione uma página chamada "home"
    • Copie o código de integração da página
    • Abra o arquivo que está visualizando agora 'starterkit/index.html'
    • Cole o código da página em qualquer lugar do html
  3. Integrando o espaço
    • De volta ao Adlayer, adicione um espaço publicitário chamado "fullbanner" na página "home"
    • Copie o código do espaço publicitário
    • Em seu arquivo 'index.html' localize onde o espaço deve ser inserido (provavelmente na linha 68) e cole o código do espaço
  4. Vinculando a campanha
    • Adicione uma campanha de nome 'Respeito pelo seu site', para o anunciante 'Adlayer'
    • Deixe a campanha iniciar agora
    • Adicione a peça 'fullbanner image' para a a campanha ''Respeito pelo seu site' fazendo upload do arquivo arquivos/fullbanner.png
    • Vincule a peça no espaço publicitário 'fullbanner' da página 'index'
  5. seu browser, e pronto.

Se você seguiu todos os passos você deverá ver a peça no espaço acima
