Adlayer Starterkit

Kit para ajudar você a aprender como integrar o Adlayer.
  • Faça download do kit
  • Extraia os arquivos no seu webserver
  • E descubra como é fácil integrar o Adlayer em suas páginas.

Dúvidas

Adlayer suporte (contato@adlayer.org)

Download

Você pode fazer download em formato zip ou tar

Clone do projeto

Se preferir também pode ter um cópia deste kit usando Git por rodar: 

$ git clone git://github.com/adlayer/starterkit

obtenha o código fonte no GitHub : adlayer/starterkit
